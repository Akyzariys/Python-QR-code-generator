# Python-QR-code-generator
Creates QR codes from text or links.

## Installation

Clone from git:
```bash
git clone https://github.com/Akyzariys/Python-QR-code-generator.git
```

Go to the folder:
```bash
cd QR-code-generator
```

Installing dependencies:
```bash
pip install -r requirements.txt
```

## Launch

```bash
python3 qr.py
```
